# Challenge 2 TP3 - Footwear Store App

A modern Android application for a footwear store, built with Jetpack Compose following a specific Figma design.

## 🚀 Features

- **Home Screen**: Browse a catalog of high-quality boots with detailed cards.
- **Product Detail**: Select sizes and quantity for your favorite products.
- **Favorites**: Manage your wishlist with a clean, numbered list interface.
- **Profile**: View and edit user information with a custom UI.
- **Settings**: Toggle notifications, dark mode, and access app policies.
- **Custom Navigation**: 
    - **Floating Bottom Bar**: Unique "mountain notch" design with a central action button.
    - **Navigation Drawer**: Side menu for quick access to all sections.

## 🛠️ Technologies

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Design System**: [Material 3](https://m3.material.io/)
- **Navigation**: [Navigation Compose](https://developer.android.com/jetpack/compose/navigation)
- **Theme**: Custom implementation of colors (`PrimaryBrown`, `BackgroundBeige`, etc.) and shapes.

## ⚙️ Setup

1. Clone the repository.
2. Open the project in **Android Studio Hedgehog** or newer.
3. Sync Gradle and run the `app` module on an emulator or physical device.
